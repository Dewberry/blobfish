Writing CKAN extension in docker container
- Docker compose up
	docker compose -f docker-compose.dev.yml up
- Docker compose execute command
	docker compose -f docker-compose.dev.yml exec ckan-dev /bin/sh -c "ckan generate extension --output-dir /srv/app/src_extensions"
- Make edits to plugin
- Register plugin in setup.py (like example shown below, but replace extension name and plugin class name
	entry_points="""
        	[ckan.plugins]
        	example_newextension=ckanext.newextension.plugin:ExampleNewExtensionPlugin
    	"""
- Add registered name to .env variable CKAN__PLUGINS
	CKAN__PLUGINS = envvars ... example_newextension
