Description

SourceDataset is the fundamental source file for all SourceGrid's used to develop the CompositeGrid.

Alll SourceDatasets are zipfiles (available at http://hydrology.nws.noaa.gov/pub/aorc-historic/) which contain hourly netcdf files (.nc4).

SourceDatasets are named using the convention "[RFCAlias]RFC_YYYYMM.zip" beginning on 1979-02.
